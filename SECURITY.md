# Security Policy

The following table shows which versions receive security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

The mishmash io team appreciates your efforts to responsibly disclose your
findings, and will make every effort to acknowledge your contributions.

To report a security issue, please use the GitHub Security Advisory
["Report a Vulnerability"](https://github.com/mishmash-io/setup-telemetry-collection/security/advisories/new)
tab.

We will then send a response indicating the next steps in handling your report.
After the initial reply to your report, the security team will keep you informed
of the progress towards a fix and full announcement, and may ask for additional
information or guidance.
